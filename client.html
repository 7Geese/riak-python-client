RiakClient¶

class riak.client.RiakClient(host='127.0.0.1', port=8098, prefix='riak', mapred_prefix='mapred', transport_class=None, client_id=None)¶

The RiakClient object holds information necessary to connect to Riak. The Riak API uses HTTP, so there is no persistent connection, and the RiakClient object is extremely lightweight.

Construct a new RiakClient object.

Parameters:
  • host (string) – Hostname or IP address
  • port (integer) – Port number
  • prefix (string) – Interface prefix
  • mapred_prefix (string) – MapReduce prefix
  • transport_class (RiakTransport) – transport class to use

set_r(r)¶

Set the R-value for this RiakClient. This value will be used for any calls to RiakBucket.get or RiakBucket.get_binary where 1) no R-value is specified in the method call and 2) no R-value has been set in the RiakBucket.
